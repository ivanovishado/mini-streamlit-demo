Contexto antes de codigo
02
Definicion simple

Streamlit convierte Python en una interfaz web

No empezamos con HTML, CSS o JavaScript. Empezamos con el script que ya entiende el equipo.

La promesa

  • Escribes Python.
  • Agregas widgets.
  • Streamlit renderiza la app.
  • Iteras en segundos.
Script primero, app despues
03
Modelo mental

Cada interaccion vuelve a correr el script

1

Usuario cambia widget

Slider, selectbox, checkbox o boton.

2

Streamlit rerun

El archivo se ejecuta de arriba hacia abajo.

3

Python calcula

Pandas filtra, agrupa y resume.

4

La pantalla cambia

Tablas, metricas y graficas se actualizan.

Si entiendes reruns, entiendes Streamlit
04
Primer comando

La app empieza con un archivo Python normal

import streamlit as st

st.title("Mi primera app")
st.write("Hola, datos")

Ejecutar

En terminal:

streamlit run streamlit_app.py
Dia 1: primer run
05
Componentes UI

Los widgets son variables con interfaz

Texto

st.title, st.write, st.markdown explican el analisis.

Seleccion

st.selectbox y st.multiselect acotan categorias.

Rangos

st.slider convierte umbrales en decisiones visibles.

Acciones

st.button y st.download_button disparan momentos concretos.

El widget regresa un valor de Python
06
Patron central

Widget -> variable -> filtro -> resultado

min_grade = st.slider(
    "Calificacion minima",
    0, 100, 80
)

filtered = data[
    data["calificacion"] >= min_grade
]

st.dataframe(filtered)

No es magia de frontend. Es Pandas reaccionando a valores que una persona controla.

10
Carga y cache

No recargues datos pesados en cada rerun

@st.cache_data
def get_students():
    return load_students(DATA_PATH)

data = get_students()

Cachear no es optimizacion prematura cuando el archivo crece: es evitar que la app se sienta lenta desde el primer filtro.

Cache para datos, estado para interaccion

13
Errores esperados

Problemas que conviene normalizar desde el inicio

Ruta del CSV

Usar Path evita depender de desde donde se corre la terminal.

Tipos de datos

Un numero leido como texto rompe filtros silenciosamente.

Reruns

Si una variable no persiste, revisar cache o session state.

Que falle rapido y con explicacion
