# Solucion de referencia

La solucion completa esta implementada directamente en la app principal:

```bash
streamlit run streamlit_app.py
```

Archivos clave:

- `streamlit_app.py`: navegacion multipagina.
- `src/streamlit_course/app_pages.py`: paginas de la app.
- `src/streamlit_course/data_utils.py`: carga, filtros y resumen de datos.

Para adaptar esta solucion al material real de Semana 3, reemplaza el CSV y ajusta las columnas esperadas en `data_utils.py`.
