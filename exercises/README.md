# Ejercicios

Corre cada ejercicio con:

```bash
streamlit run exercises/day1/01_first_app.py
```

La ruta cambia segun el archivo que quieras abrir.

## Dia 1

- `day1/01_first_app.py`: crear la primera app.
- `day1/02_widgets.py`: conectar sliders y selectbox con texto.

## Dia 2

- `day2/01_pandas_filters.py`: filtrar un dataframe con un slider.
- `day2/02_capstone_starter.py`: completar el dashboard final.

La app completa de referencia esta en `streamlit_app.py`.
